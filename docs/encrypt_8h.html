Encrypts the given data using ECIES over Curve25519 and AES256-GCM.

Parameters
dataThe data to encrypt.
data_lengthThe length of the data array.
public_keyThe public key to encrypt the data with (hex-string format, as is the output of cecies_generate_curve25519_keypair()).
outputWhere to write the encrypted output into (please pre-allocate this big enough, you can use cecies_curve25519_calc_output_buffer_needed_size() to find out how much you need).
output_bufsizeHow big the output buffer is.
output_length[OPTIONAL] Where to write the output buffer length into (this will be less than the provided output_bufsize). Pass NULL if you don't care (e.g. you already know the output size because you allocated space using cecies_curve25519_calc_output_buffer_needed_size()).
output_base64Should the encrypted output bytes be base64-encoded for easy transmission over e.g. email? If you decide to base64-encode the encrypted data buffer, please be aware that a NUL-terminator is appended at the end to allow usage as a C-string but it will not be counted in output_length.
Returns
0 if encryption succeeded; error codes as defined inside the header file or MbedTLS otherwise.

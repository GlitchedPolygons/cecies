Encrypts the given data using ECIES over Curve448 and AES256-GCM.

Parameters
dataThe data to encrypt.
data_lengthThe length of the data array.
public_keyThe public key to encrypt the data with.
public_key_lengthLength of the public_key string.
public_key_base64Is the public_key base64-formatted? If this is false, the key will be directly fed into mbedtls_ecp_point_read_binary()!
outputWhere to write the encrypted output into (please pre-allocate this big enough).
output_bufsizeHow big the output buffer is.
output_lengthWhere to write the output buffer length into (this will be less than the provided output_bufsize).
output_base64Should the encrypted output bytes be base64-encoded for easy transmission over e.g. email? If you decide to base64-encode the encrypted data buffer, please be aware that a NUL-terminator is appended at the end to allow usage as a C-string but it will not be counted in output_length.
Returns
0 if encryption succeeded; error codes as defined inside the header file or MbedTLS otherwise.
